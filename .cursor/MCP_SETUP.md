# MCP 서버 설정 가이드

이 문서는 Cursor IDE의 MCP (Model Context Protocol) 서버 설정 방법을 설명합니다.

## 현재 설정된 MCP 서버

### ✅ AWS Documentation Server
- **명령어**: `uvx awslabs.aws-documentation-mcp-server@latest`
- **설명**: AWS 문서를 검색하고 참조할 수 있는 MCP 서버
- **상태**: 설정 완료

## 브라우저 확장 기능 설정

Cursor의 브라우저 확장 기능을 사용하면 AI가 브라우저를 직접 제어할 수 있습니다.

### 브라우저 확장 기능 활성화 방법

1. **Cursor 확장 기능 설치**
   - Cursor IDE에서 확장 기능 마켓플레이스 열기
   - "Browser Extension" 또는 "Cursor Browser" 검색
   - 설치 및 활성화

2. **브라우저 확장 기능 확인**
   - 브라우저 확장 기능이 설치되면 자동으로 MCP 서버로 등록됩니다
   - 별도의 `mcp.json` 설정이 필요하지 않을 수 있습니다

3. **브라우저 확장 기능 사용**
   - 브라우저 관련 작업 요청 시 자동으로 사용됩니다
   - 예: "이 웹사이트를 열어서 확인해줘", "브라우저에서 검색해줘"

### 브라우저 확장 기능이 작동하지 않는 경우

1. **확장 기능 설치 확인**
   - Cursor 설정에서 확장 기능 목록 확인
   - 브라우저 확장 기능이 설치되어 있는지 확인

2. **Cursor 재시작**
   - 확장 기능 설치 후 Cursor를 완전히 재시작

3. **MCP 서버 연결 확인**
   - Cursor 설정에서 MCP 서버 상태 확인
   - 브라우저 확장 기능 MCP 서버가 연결되어 있는지 확인

## MCP 서버 추가 방법

새로운 MCP 서버를 추가하려면 `~/.cursor/mcp.json` 파일을 수정하세요:

```json
{
  "mcpServers": {
    "서버 이름": {
      "command": "실행 명령어",
      "env": {
        "환경변수": "값"
      },
      "args": []
    }
  }
}
```

### 예시: 브라우저 MCP 서버 추가 (필요시)

일부 브라우저 MCP 서버는 다음과 같이 설정할 수 있습니다:

```json
{
  "mcpServers": {
    "Browser": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-browser"],
      "env": {}
    }
  }
}
```

**참고**: Cursor의 브라우저 확장 기능은 일반적으로 별도 설정 없이 작동합니다.

## MCP 서버 연결 문제 해결

### 문제: MCP 서버가 연결되지 않음

1. **명령어 확인**
   - `mcp.json`의 `command`가 올바른지 확인
   - 터미널에서 직접 실행하여 작동하는지 확인

2. **의존성 확인**
   - 필요한 도구가 설치되어 있는지 확인 (예: `uvx`, `npx`, `node`)
   - 환경 변수가 올바르게 설정되어 있는지 확인

3. **로그 확인**
   - Cursor 설정에서 MCP 서버 로그 확인
   - 오류 메시지 확인

### 문제: 브라우저 확장 기능이 작동하지 않음

1. **확장 기능 설치 확인**
   - Cursor 확장 기능이 설치되어 있는지 확인
   - 브라우저 확장 기능이 활성화되어 있는지 확인

2. **권한 확인**
   - 브라우저 확장 기능이 필요한 권한을 가지고 있는지 확인

3. **Cursor 재시작**
   - 확장 기능 설치 후 Cursor를 완전히 재시작

## 현재 MCP 설정 파일 위치

- **전역 설정**: `~/.cursor/mcp.json`
- **프로젝트별 설정**: 프로젝트별 MCP 설정은 지원되지 않습니다 (전역 설정만 사용)

## 참고사항

- MCP 서버 설정을 변경한 후에는 Cursor를 재시작해야 할 수 있습니다
- 브라우저 확장 기능은 Cursor 확장 기능으로 제공되므로 별도 설정이 필요하지 않을 수 있습니다
- MCP 서버 연결 문제가 지속되면 Cursor 로그를 확인하세요
